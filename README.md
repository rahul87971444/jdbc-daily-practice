# 🚀 JDBC Daily Practice 

A structured **20-Day JDBC Learning Journey** designed to strengthen Java Database Connectivity (JDBC) concepts through daily hands-on coding exercises.

## 📚 Overview

This repository contains practical JDBC programs covering fundamental to intermediate database operations such as:

* Database connections
* ResultSet handling
* Transactions & Savepoints
* Prepared Statements
* CRUD Operations
* Metadata APIs
* Views & Indexes
* Batch Processing
* Stored Procedures
* Connection Pooling
* Student Management System Project

Each day focuses on a specific JDBC concept, helping developers build strong database programming skills with Java.

* MySQL Server installed
* MySQL JDBC Driver (`mysql-connector-j`)
* IDE (Eclipse / VS Code)
